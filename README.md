# Events App

A full-stack application for managing and browsing events.

## Project Structure

```
events-app/
├── backend/          # Node.js/Express server
│   └── src/
│       ├── routes/   # API routes
│       ├── controllers/  # Business logic
│       └── models/   # Data models
└── frontend/         # React application
    └── src/
        ├── components/  # Reusable components
        └── pages/   # Page components
```

## Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
